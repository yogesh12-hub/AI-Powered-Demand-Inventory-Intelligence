# AI-Powered Demand & Inventory Intelligence Platform

## Project Overview
This project focuses on building an AI-powered platform for demand forecasting and inventory optimization using machine learning techniques.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess data
- Engineer meaningful features
- Forecast demand using Prophet and LSTM
- Segment customers
- Optimize inventory
- Build an interactive Streamlit dashboard
- Deploy the application

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Prophet
- PyTorch
- Streamlit

## Dataset
Online Retail Dataset
