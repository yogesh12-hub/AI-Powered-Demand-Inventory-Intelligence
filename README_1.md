# 📊 RetailPulse – AI-Powered Demand & Inventory Intelligence Platform

## 📌 Project Overview

RetailPulse is an end-to-end Data Science project that helps retail businesses make data-driven decisions using customer analytics and demand forecasting.

The project performs customer segmentation, predicts future demand, analyzes customer churn, and provides inventory recommendations through an interactive Streamlit dashboard.

---

## 🚀 Features

- 📈 Data Cleaning & Preprocessing
- 📊 Exploratory Data Analysis (EDA)
- 👥 Customer Segmentation using RFM Analysis & K-Means Clustering
- 📅 Demand Forecasting using Prophet
- ⚠️ Customer Churn Prediction
- 📦 Inventory Optimization
- 📉 Interactive Streamlit Dashboard

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/AI-Powered-Demand-Inventory-Intelligence.git
```

Move to the project folder

```bash
cd AI-Powered-Demand-Inventory-Intelligence
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
